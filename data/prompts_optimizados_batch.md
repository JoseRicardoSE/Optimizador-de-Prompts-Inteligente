## Idea Original: quiero hacer una app de js, que necesito para que sea moderna etc


**Tono:** Profesional

### ✨ Prompt Optimizado
```text
Eres un experto en desarrollo web.

Objetivo: Crear una aplicación web moderna utilizando JavaScript.

Requisitos:
Identifica los componentes necesarios para construir una aplicación web moderna, incluyendo la elección de frameworks, bibliotecas y tecnologías.

Instrucciones:
Análiza los siguientes aspectos y proporciona una lista de herramientas y tecnologías adecuadas para desarrollar una aplicación web moderna:

* Frameworks de frontend
* Bibliotecas para gestión de estado y routing
* Motor de renderizado
* Herramientas de build y deployment

Considera las siguientes preguntas al responder:

* ¿Qué características de una aplicación web moderna son esenciales para su éxito?
* ¿Qué tecnologías y herramientas se recomiendan para lograr la eficiencia y escalabilidad?
```

**Variables:** -

**Explicación:** Se transformó el prompt original en un prompt profesional, dividido en secciones claras y organizadas, con instrucciones explícitas para el modelo y una breve explicación de las técnicas utilizadas.

## Idea Original: como hacer un gato con sobrero en un gemerador de imagenes


**Tono:** Profesional

### ✨ Prompt Optimizado
```text
Eres un experto en diseño de imágenes.

Contexto:
El objetivo es crear un gato con sobrero en un generador de imágenes.

Instrucciones:
1. Define los elementos necesarios para la creación del gato con sobrero (p. ej., forma del gato, color, detalles del sobrero).
2. Selecciona un generador de imágenes adecuado para la creación de la imagen.
3. Diseña el gato con sobrero siguiendo los elementos definidos, teniendo en cuenta las limitaciones del generador de imágenes.

Piensa estratégicamente sobre las decisiones de diseño para obtener un resultado exitoso.
```

**Variables:** NINGUNA

**Explicación:** Se optimizó el prompt para un tono profesional, se agregó un contexto claro y se establecieron instrucciones específicas para la tarea. Se eliminaron etiquetas XML y se reemplazaron con párrafos limpios.
